# Juno DS-88 Control Center

Desktop software project for improving interaction with the Roland Juno DS-88 during study, sound design, patch management, and live performance.

## Project purpose

Juno DS-88 Control Center is a personal desktop application built to make working with a Roland Juno DS-88 more practical and organized.

The long-term goal is to create a tool that helps with:

- patch and performance management;
- sound organization;
- local preset storage;
- dual/split setup management;
- studio and live workflow support;
- future MIDI-based interaction and control features.

## Current focus

The project is currently in the initial setup phase.

The current priority is to build a solid technical foundation before implementing keyboard-specific features:

- development environment setup;
- repository structure;
- JUCE + CMake integration;
- first compilable desktop application;
- gradual architecture definition.

## Tech stack

- C++20
- JUCE
- CMake
- Git + GitHub
- Visual Studio Build Tools / MSVC
- VS Code
- Windows as primary development environment

## Development approach

The project is being built with a focus on:

- learning the stack deeply, not just assembling code;
- keeping the architecture simple and scalable;
- separating concerns progressively;
- avoiding premature overengineering;
- growing toward a serious and maintainable desktop application.

## Planned areas

The project will likely evolve around these areas:

- application UI;
- MIDI communication;
- patch/performance domain modeling;
- local persistence;
- setup organization for studio and live usage.

## Status

Early setup and repository bootstrap.
